
# Number Guessing Game in C

This is a simple menu-driven Number Guessing Game developed using the C programming language.

## Features
- Menu option to start or exit the game
- User inputs a number and gets hints
- Displays whether the guess is too high or too low
- Continues until the correct number is guessed

## Concepts Used
- Loops (while)
- Conditional statements (if-else)
- User input using scanf and printf
- Basic logic building

## Learning Outcome
This project helped me understand:
- Control flow in C
- Writing menu-driven programs
- Handling user input and conditions
